p4a Python Package Index

This is a PEP 503 “simple” Python package index. It is intended for use with pip, not for browsing.

To install packages from this index:

pip install <package> \
  --index-url https://<user>.github.io/<repo>/simple \
  --trusted-host <user>.github.io

You may also use it as a fallback alongside PyPI: 

pip install <package> \
  --extra-index-url https://<user>.github.io/<repo>/simple

The package indexes themselves live under: /simple/<package-name>/
